# Whitecosmos
A whitespace interpreter (and hopefully compiler and transpiler oneday) written in Rust

## Usage

### Run a program
- Run a whitespace program
```bash
whitecosmos file.ws
```

- Run a fakewhitespace program
```bash
whitecosmos file.fws
```

- Run a visiblewhitespace program
```bash
whitecosmos file.vws
```
